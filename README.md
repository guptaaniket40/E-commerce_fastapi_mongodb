# 🛒 FastAPI E-commerce API (MongoDB)

Simple eCommerce backend using **FastAPI + MongoDB (MongoEngine)**.

## 🚀 Features

* Product CRUD (Create, Read, Update, Delete)
* Cart (Create, Add, View, Remove/Decrease)
* Checkout & Dummy Payment

## ⚙️ Setup

```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
```

## 🌐 API Docs

```bash
http://127.0.0.1:8000/docs
```

## 📌 Note

* No authentication
* Single cart system
* Dummy payment only
